Step 1: Find Android SDK Location
  - Open Android Studio
  - Go: Tools â†’ SDK Manager
  - Look at "Android SDK Location"
  - Example: C:\Users\YourName\AppData\Local\Android\Sdk

Step 2: Add to PATH
  - Press: Windows + X
  - Select: System (Settings)
  - Click: Advanced system settings (left side)
  - Click: Environment Variables button
  - Under "User variables", select: Path
  - Click: Edit
  - Click: New
  - Paste: C:\Users\YourName\AppData\Local\Android\Sdk\platform-tools
  - Click: OK, OK, OK

Step 3: Restart Terminal/CMD
  - Close all cmd/PowerShell windows
  - Open new cmd window
  - Type: adb devices
  - Should show your emulator/device
